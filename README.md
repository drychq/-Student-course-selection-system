# Student-course-selection-system


## 简介
这是一个基于命令行的学生选课系统。

使用c++完成，仅支持linux系统。

## 部署步骤
1. 克隆项目：
git clone https://github.com/your-repo/StudentCourseSystem.git 

cd Student-course-selection-system

git switch release

2. 运行安装脚本：
chmod +x deploy/install.sh

./deploy/install.sh

3. 打开系统菜单，搜索 Student-course-selection-system并运行。
